Workin - Política de privacidade

Este aplicativo recebe alguns dados pessoais de seus usuários para uso em ferramentas de criação de currículo e incrição de vagas de emprego, os dados disponibilizados pelos usuários da plataforma pode ser utilizados em também em plataformas de estatísticas e de anuncio segmentados para fins de monetização e evolução das ferramentas do próprio aplicativo.

Os Dados Pessoais são coletados para os seguintes propósitos e usando os seguintes serviços:

Estatísticas

Google Analytics: Dados Pessoais: Cookie; Dados de uso.

Testes Beta

- Central de aplicativos: Dados Pessoais: e-mail; Todos os dados que compartilham durante o uso do serviço; vários tipos de dados como política de privacidade do serviço. - Teste beta do Google Play: Dados Pessoais: data de nascimento; o email; informações do dispositivo; nome; nome de Usuário; país; sexo; sobrenome; Todos os dados que compartilham durante o uso do serviço; vários tipos de dados como política de privacidade do serviço
